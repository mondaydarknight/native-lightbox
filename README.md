# Native-Lightbox

### Lightbox UI with native JavaScript
Lightbbox is native JavaScipt library code used to overlay images on top of the current page. It's a snap to setup and works on all modern browsers.

## Getting Started
Make sure you have npm installed on your machine.
- Clone the project from repository [github.com/mongcheng/angular1-es6-phpslim](https://github.com/mongcheng/native-lightbox)
- In your terminal, cd into the cloned folder and run `npm install`.
- Run`webpack`. This allows you view file changes you make.
